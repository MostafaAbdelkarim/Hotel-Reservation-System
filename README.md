## Basic Overview

Backend API based on Spring Boot and related Spring frameworks for managing hotel reservations.

<br>

## Tech Stack
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL/H2-Database
- ThymeLeaf
- Swagger Documentation
- Lombok

## Dependencies
You can find all dependencies in `pom.xml` file

## How to use
- Clone the repo.
- Run using IntelliJ IDEA
- Connect datasource to your Postgres/H2 database by modifying application.properties file.
- You can start experimenting using this link:
```text
http://localhost:8080/
```
